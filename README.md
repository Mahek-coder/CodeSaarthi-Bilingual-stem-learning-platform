#  CodeSaarthi  
*A Bilingual AI-powered DSA Learning Platform (Hindi + English)*

---

##  Problem Statement & Solution Overview

### ❗ Problem Statement
- Data Structures and Algorithms (DSA) is difficult for beginners  
- Most learning resources are available only in English  
- Students lack structured and guided learning paths  
- Concept understanding becomes confusing without proper explanation  

### Solution Overview
CodeSaarthi is an AI-powered learning platform designed to simplify DSA learning by:
- Providing bilingual explanations in Hindi and English  
- Offering AI-based doubt solving support  
- Generating personalized learning roadmaps  
- Including interactive quizzes for practice  

---

## Tech Stack

- Frontend: React / Vite  
- Backend: Firebase  
- Database: Firestore  
- Authentication: Firebase Authentication  
- AI Integration: Google Gemini API / OpenAI API  
- Hosting: Firebase Hosting  

---

## ⚙️ Installation & Setup Instructions

```bash
# Clone the repository
git clone https://github.com/your-username/codesaarthi.git

# Move into project directory
cd codesaarthi

# Install dependencies
npm install

# Create .env file and add keys
VITE_FIREBASE_API_KEY=your_firebase_key
VITE_AI_API_KEY=your_ai_key

# Run the project
npm run dev
